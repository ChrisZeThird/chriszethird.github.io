About this bot!

BotZeFourth is designed for artists. It automatically creates threads for artwork posts, keeping discussions separate from main chats. Artists can also create, modify, or delete information cards, and the bot offers templates for basic and D&D-style character sheets.

Slash commands

Thanks to slash commands, it is much easier to see every single command the bot has to offer by simply typing `/` in any channel! You can also use the help commands to see the full list.
